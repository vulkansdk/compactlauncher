Compact Launcher
=======

Compact Launcher is an open-source Minecraft launcher focused on reliability, speed and simplicity.

The launcher is written in C++ and uses the Qt framework for the user interface.
The goal of the project is to keep the launcher lightweight, predictable and easy to maintain.

Compact Launcher is designed to be fast and compact, avoiding unnecessary features and complexity while still providing the essential functionality needed to launch and manage Minecraft instances.

## Contributing
If you want to contribute to the project, please join the [Discord](https://discord.gg/GX3EsZ7UBK) server first and discuss your idea there.

Pull requests that were not discussed beforehand may be rejected.
This helps keep the project consistent and avoids unnecessary changes.
